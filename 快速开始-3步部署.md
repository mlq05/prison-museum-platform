# ⚡ 快速开始 - 3步部署

## 第一步：生成 ZIP 包

打开 CMD，执行：

```cmd
cd /d F:\test4\服务器
打包-修复版.bat
```

等待生成 `server.zip` 文件。

---

## 第二步：上传到云开发控制台

1. **访问控制台**：https://console.cloud.tencent.com/tcb
2. **选择环境**：`test-7g8oe7lq75832d7c`
3. **进入云托管**：左侧菜单 → "云托管" → "服务管理"
4. **创建服务**（如果没有）：
   - 点击"新建服务"
   - 名称：`museum-api`
   - 提交
5. **上传代码**：
   - 点击服务进入详情
   - 点击"新建版本"
   - 选择"本地上传"
   - 上传 `F:\test4\服务器\server.zip`
6. **配置环境变量**：
   ```
   NODE_ENV=production
   JWT_SECRET=你的随机密钥（至少32位）
   PORT=80
   ```
7. **点击"部署"**

---

## 第三步：获取服务地址

部署成功后，在服务详情页获取地址：
```
https://museum-api-xxxxx.tcb.qcloud.la
```

测试访问：`https://museum-api-xxxxx.tcb.qcloud.la/health`
