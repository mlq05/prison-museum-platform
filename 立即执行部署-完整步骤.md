## 现在需要做的

### 方法一：使用 CMD 执行（推荐）

1. **打开 CMD（命令提示符）**：
   - 按 `Win + R`
   - 输入 `cmd`
   - 回车

2. **切换到项目目录**：
   ```cmd
   cd /d F:\test4\服务器
   ```

3. **执行部署命令**：
   ```cmd
   cloudbase run deploy -e test-7g8oe7lq75832d7c -s museum-api --containerPort 80
   ```

**注意**：
- 如果服务 `museum-api` 不存在，CLI 会提示是否创建，输入 `y` 确认
- 部署过程需要几分钟，请耐心等待

## 部署成功后

1. **获取服务地址**：
   - 登录云开发控制台：https://console.cloud.tencent.com/tcb
   - 选择环境：`test-7g8oe7lq75832d7c`
   - 进入"云托管" → "服务管理" → `museum-api`
   - 查看服务地址（类似：`https://museum-api-xxxxx.tcb.qcloud.la`）

2. **配置环境变量**（在控制台）：
   ```
   NODE_ENV=production
   JWT_SECRET=你的随机密钥（至少32位字符）
   PORT=80
   ```

3. **测试服务**：
   - 访问：`https://museum-api-xxxxx.tcb.qcloud.la/health`
   - 应该返回 JSON 响应
